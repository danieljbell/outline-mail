## Dependencies
Node.js

## Quick Start
- Clone the repository
  ```
  [sudo] git clone git@github.com:matt-harris/outline-mail.git
  ```
- Install depedencies using NPM
  ```
  npm install
  ```
- Run Gulp's default task
  ```
  gulp
  ```
